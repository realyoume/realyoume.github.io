项目结构

每个模块都相当于一个小项目

xuecheng-plus-parent

所有模块的父模块，主要用于版本管理

通过 dependencyManagement 锁定项目依赖的版本

xuecheng-plus-base

包含一些通用的代码

其他模块通过 maven 引入

xuecheng-plus-content

具体的功能模块

MybatisPlus

可以根据数据库结构生成实体类，持久层，业务层，接口层代码

分页查询

配置类

通过条件和分页参数进行查询

占位符

#{} 占位符，可以防止 sql 注入

${} 动态拼接字符串，会导致 sql 注入

Swagger

用于编写接口文档

可以读取代码生成文档，并直接测试

数据字典表

用于定义状态与状态码之间的映射关系

方便之后的修改和维护

单元测试

每写一个类就要有一个单元测试

HttpClient

IDEA 自带的插件，可以方便的发送请求测试接口

请求数据用文件保存，可以与项目文件同时存储

跨域

浏览器的安全机制，同源策略

服务端允许跨域

树形结构

关键参数

id parentid

递归查询语句

list 2 map

创建对象

  1. 参数合法性校验
  2. @Transactional

Beanutils.copyProperties

拷贝属性名相同的属性

注意源和目标

统一异常处理

@ControllerAdvice

@ExeceptionHandler

自定义异常类型

业务代码抛出异常

异常处理类捕获异常

返回前端异常返回类

参数校验

JSR 303 校验

@RequestBody @Validated

开启对对象的参数校验

当参数非法时，会报出 MethodArgumentNotValidException 错误，并携带信息

可以通过统一异常处理进行管理，将报错返回给前端

分组

参数校验可以分组进行

同一个对象在不同情况下使用不同的校验规则

可以通过 groups 属性来配置
