什么是Redis

一种数据库，特点是key-value,开源，使用C语言编写

重要文件

  • redis-server 服务端
  • redis-cli 客户端
  • redis.conf 配置内容

常用数据类型

  • 字符串 string
  • 哈希 hash
  • 列表 list
  • 集合 set
  • 有序集合 sorted set

常用指令
